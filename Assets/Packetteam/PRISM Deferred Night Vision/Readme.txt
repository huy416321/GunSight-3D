Welcome to PRISM Deferred Night Vision, a deferred Night Vision effect for Unity URP and Render Graph!

Quickstart guide:
1. Open your URP Renderer asset and add PRISM/PRISM Deferred Night Vision as a new Renderer Feature.
2. Add PRISM Deferred Night Vision as a component to a global volume in your scene.
3. The default settings should now be showing in the Game View camera. 
4. Tweak settings as required.

NOTE: Requires deferred rendering path to work!

Check out PRISM on the Unity Asset Store: https://assetstore.unity.com/packages/vfx/shaders/fullscreen-camera-effects/prism-2-realistic-post-processing-157871
